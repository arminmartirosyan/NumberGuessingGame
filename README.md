#NumberGuessingGame
#GUESS THE NUMBER

Modern desktop number guessing game built with Python and CustomTkinter.

---

## Description

GUESS THE NUMBER is an interactive desktop game where the player must guess a randomly generated number within a selected range.  
The project features a modern graphical user interface, multiple difficulty levels, real-time feedback, score calculation, timer tracking, hints, and persistent leaderboard storage.

The application was developed using Object-Oriented Programming principles and focuses on creating a clean and responsive user experience.

---

## Features

- Modern dark-themed GUI
- Multiple difficulty levels
- Real-time feedback system
- Score calculation based on performance
- Timer and attempt tracking
- Hint generation system
- Persistent leaderboard using JSON
- Input validation and error handling
- Responsive desktop interface
- Object-Oriented Programming architecture

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/guess-the-number.git
```

Move into the project folder:

```bash
cd guess-the-number
```

Install dependencies:

```bash
pip install customtkinter
```

Run the project:

```bash
python main.py
```

---

## Gameplay

1. Select a difficulty level.
2. Enter a number guess.
3. Receive instant feedback:
   - Too High
   - Too Low
   - Correct Guess
4. Use hints if needed.
5. Try to finish with the highest possible score.

The final score depends on:
- Number of attempts
- Time taken
- Difficulty level
